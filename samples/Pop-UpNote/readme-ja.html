付箋

１．このサンプルについて

付箋のサンプルです。ドラッグが出来ます。 PopUpNote-ja.html を開けば実行できます。

２．コンパイル方法

コンパイルするには、JDK 1.2 以降が必要です。

手動でコンパイルするには、以下のようにコマンドを入力してください。 現在のフォルダは、samplesとします。classes にはコンパイル済みのクラスがあります。

cd src
javac -bootclasspath ..\..\..\lib\ortoClasses.jar jp\orto\popupnote\*.java

生成されるクラスファイルから PopUpNote-ja.html を作る必要があります。それは以下のようにやります。

java -jar ..\..\..\bin\ortoar.jar -main jp.orto.popupnote.PopUpNote -mainHTML ..\design\design-ja.html -encoding Shift_JIS jp\orto\popupnote\*.class
copy PopUpNote.html ..\PopUpNote-ja.html

３．実行方法

PopUpNote-ja.html をブラウザから開くだけですが、 実行時に design フォルダを読みに行くので、PopUpNote-ja.html と design フォルダを 同じフォルダ内におく必要があります。
