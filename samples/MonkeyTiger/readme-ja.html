サルトラ算

１．このサンプルについて

このサンプルは、サルとトラが足し算をしてくれる Web アプリケーションです。 MonkeyTiger-ja.html を開けば実行できます。

２．コンパイル方法

コンパイルするには、JDK 1.2 以降が必要です。

手動でコンパイルするには、以下のようにコマンドを入力してください。 現在のフォルダは、samplesとします。classes にはコンパイル済みのクラスがあります。

cd src
javac -bootclasspath ..\..\..\lib\ortoClasses.jar jp\orto\monkeytiger\*.java

生成される２つのクラスファイルから MonkeyTiger-ja.html を作る必要があります。それは以下のようにやります。

java -jar ..\..\..\bin\ortoar.jar -main jp.orto.monkeytiger.MonkeyTiger -mainHTML ..\design\design.html -encoding Shift_JIS jp\orto\monkeytiger\*.class
copy MonkeyTiger.html ..\MonkeyTiger-ja.html

３．実行方法

MonkeyTiger-ja.html をブラウザから開くだけですが、 実行時に design フォルダを読みに行くので、MonkeyTiger-ja.html と design フォルダを 同じフォルダ内におく必要があります。
